# 🛒 Customer Segmentation & 📈 Sales-Trend Analysis for Retail-Optimization

This project delivers a complete data analytics pipeline to help a retail company understand customer behavior, identify purchase trends, and optimize its marketing and sales strategy.
Using Python, SQL, and Power BI, the analysis explores factors influencing buying decisions (discounts, seasons, reviews, payment preferences), identifies high-value customer segments, and uncovers sales trends across channels and product categories.
This project is ideal for showcasing full-stack analytics capability — from raw data to insights and dashboards.

1. Data Preparation (Python):
Cleaned raw data. Standardized product categories & demographic fields removed duplicates, treated missing values. Created new fields (total spend, discount %, review bands, etc.)
Exported final cleaned dataset for SQL

2. Data Modeling & SQL Analysis:
Designed SQL tables & relationships. Loaded cleaned data into SQL DB(PostgreSql).
  2.1 Performed analytical SQL queries:
         Customer segmentation, Repeat purchase patterns, Seasonal sales trends, Category performance, Discount effectiveness.
   
3. Power BI Dashboard:
    Created an interactive BI dashboard with:
KPIs: Revenue, Avg Order Value, Repeat Purchase Rate,
Customer Segments: High-value, discount-sensitive, seasonal shoppers,
Sales Trends: Monthly, seasonal, category-wise,
Channel Trends: Online vs. offline performance,
Marketing Insights: Discount impact, review influence.

4️. Insights & Recommendations
Delivered a detailed PDF report containing:
Customer behavior analysis,
Product/category trend insights,
Revenue-driving factors,
Marketing strategy enhancements,
Loyalty improvement ideas.
